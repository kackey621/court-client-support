## 案件切替

1. サイドバー上部の案件名をクリック
2. ドロップダウンに案件一覧が表示される
3. 目的の案件をクリックすると即座に切替

初回起動時は「サンプル案件」が 1 件表示されます。実案件は
自身のワークフローに合わせて追加してください。各案件は個別の色で
サイドバー・インポート時に区別されます。

## 文書管理

### ファイルのインポート

**対応形式**: PDF / PNG / JPG / JPEG / GIF / WEBP / TIFF / BMP / DOCX / DOC / TXT / XLSX / CSV / ZIP

#### 方法 1: ドラッグ&ドロップ

「ドキュメント」ページ → **「+ インポート」** をクリックしてドロップゾーンを表示 → ファイルをドラッグ

#### 方法 2: ファイル選択ダイアログ

「+ インポート」→「ファイルを選択」

#### 方法 3: フォルダ一括取込

「+ インポート」→「フォルダを選択」→ フォルダ内の対応ファイルをすべて自動取込

#### 方法 4: ZIP 展開取込

`.zip` ファイルをドロップまたはダイアログで選択 → 展開してすべてのファイルを取込

> **注意**: ファイルはアプリ内にコピーされません。元のファイルパスを参照します。
> ファイルを移動・削除するとプレビューできなくなります。

インポートと同時に、対応形式（PDF / 画像）は **OCR キューへ自動投入** され、
バックグラウンドで順次テキスト抽出されます。

## OCR（文字認識）

インポート時に PDF / 画像は自動で OCR キューへ登録されます。
手動で再実行したい場合は、詳細ページの **OCR** タブから「実行」をクリック。

### 実行フロー

1. インポート直後は `待機中（キューに登録済み）`
2. 順次 `処理中` に遷移、進捗バーで進行表示
3. 完了すると `完了` バッジとテキストが表示される

### OCR エンジン

- **NDLOCR-Lite**（インストーラー同梱、日本語に最適化）
- エンジンが見つからない場合は自動でフォールバック

### OCR テキストの活用

- 「コピー」ボタンでクリップボードへコピー可能
- OCR テキストは全文検索のインデックスに自動登録
- AI 要約・AI Flash でも OCR テキストが参照される
- 「.txt」ボタンで単独のテキストファイルに保存可能

## AI 要約・AI Flash

> **前提**: 設定ページで Gemini API キーを登録してください。

### AI 要約（文書単位）

1. 文書詳細ページ → 右パネル **「AI 要約」** タブ
2. **「要約生成」** ボタンをクリック
3. Gemini が OCR テキストを読み、要約を生成
4. 「コピー」ボタンでクリップボードへ

### AI Flash（文書内 Q&A）

1. 文書詳細ページ → 右パネル **「AI」** タブ
2. 入力欄に質問を入力して送信
3. チャット形式で回答が表示される

**質問例**:
- 「この契約書の解除条件を教えてください」
- 「損害賠償の根拠条文はどれですか？」
- 「相手方の主張の問題点を挙げてください」

### AI Flash（案件全体 Q&A）

サイドバーの **「AI Flash」** ページでは、案件全体の質問が可能です。

> **プライバシー**: 送信したテキストは Gemini API の学習には使用されません。
> 機密性の高い固有名詞等は送信前に確認してください。

## 全文検索

### 検索の実行

- **方法 1**: TopBar の検索バーに入力して Enter
- **方法 2**: サイドバーの「検索」ページでタブを選び詳細検索

### 検索スコープ（タブ）

- **ドキュメント** — 案件内の文書を FTS5 で全文検索
- **法令** — e-gov 法令 API でリアルタイム検索
- **判例** — 外部判例検索サイトへのリンク生成

### 検索対象（ドキュメント）

- 文書タイトル
- OCR で抽出したテキスト
- AI 要約テキスト
- タグ

### 検索履歴・推薦クエリ

- 過去の検索クエリが自動で記録される
- 検索ページ上部に「よく使う検索」として表示
- 案件ごとに独立した検索履歴を管理

## 法令検索

### 法令の検索

1. サイドバー「法令検索」ページまたは「検索」ページの **法令** タブを開く
2. キーワードを入力して検索
3. 一致した法令の一覧が表示される
4. 法令名をクリックすると条文を展開表示

### クイック検索ボタン

よく使う法令名のボタンが用意されています：
- 民法 / 商法 / 民事訴訟法 / 労働契約法
- 会社法 / 不法行為 / 損害賠償 など

### 判例検索

「判例」タブから外部サイトへのリンク生成：
- **裁判所 公式判例検索**
- **D1-Law.com 判例検索**

### 法令テキストのコピー

条文表示後、「テキストをコピー」ボタンでクリップボードへコピー。

## メモ・Wiki / ブックマーク

### メモの作成（案件単位）

1. サイドバー「メモ・Wiki」ページを開く
2. **「+ 新規メモ」** をクリック
3. タイトル・内容・タグ（カンマ区切り）を入力
4. **「作成」** をクリック

### Wiki（文書単位）

各ドキュメント詳細ページの **「Wiki」** タブから、Markdown で
文書固有の論点整理を記述できます。

### メモの操作

| 操作 | 方法 |
|------|------|
| 編集 | メモカードをクリック |
| ピン留め | カードの 📌 アイコン（ピン留め済みは一覧最上位に表示） |
| コピー | カードの 📋 アイコン |
| 削除 | カードの 🗑️ アイコン |

### ブックマーク

1. 文書カードまたは詳細ページの ⭐ アイコンをクリック
2. サイドバー「ブックマーク」ページで一覧確認
3. ブックマーク一覧から文書詳細へ直接ジャンプ可能

## 印刷・コピー・出力

### 文書の印刷

文書詳細ページのツールバー → 🖨️ アイコン → システムの印刷ダイアログ

### テキストのコピー

| 場面 | 操作 |
|------|------|
| OCR テキスト | OCR タブの「コピー」ボタン |
| AI 要約テキスト | AI 要約タブの「コピー」ボタン |
| 法令テキスト | 条文表示後の「テキストをコピー」ボタン |
| メモ内容 | メモカードの 📋 ボタン |

### 出力（出力タブ）

文書詳細ページの **「出力」** タブから:

- **Word (.docx)** — Wiki・AI 要約・OCR・コメントを整形して書き出し
- **Google Docs 用 HTML** — クリップボードに書式付きでコピー
- **OCR テキスト (.txt)** — OCR 本文のみを単独ファイルに
- **テキスト (.txt)** — 複合テキストとして出力

### Finder で元ファイルを開く

文書詳細ページのツールバー → 📁 アイコン → 元ファイルのあるフォルダが開く

## 設定

### 設定ページへのアクセス

左サイドバー最下部の **⚙ 設定** をクリック

### 設定項目

| 項目 | 説明 |
|------|------|
| Gemini API キー | AI 機能に使用する API キー（マスク表示） |
| 使用モデル | Gemini のモデル選択 |
| 接続テスト | API キーの動作確認 |
| OCR 言語 | テキスト認識の言語設定 |
| 認証方式 | Touch ID / Windows Hello / 6 桁 PIN / なし |

### 設定の保存場所

設定はすべてローカルの SQLite データベース（`settings` テーブル）に保存されます。
アンインストール時はデータベースごと削除されます。
